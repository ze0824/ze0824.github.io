String、StringBuffer、StringBuilder的区别

String：不可变字符串类

StringBuffer：长度可变的字符串类，因为被synchronized修饰，所以线程安全。但是加锁了，所以性能偏低。

StringBuilder：也是长度可变的字符串类，性能偏高，但是线程不安全。

JVM堆

新生代包括Eden区、survivor区；老年代是old区。常量池属于方法区PermGen。

JVM提供三个ClassLoader，层级由高到低。

BootStrap classLoader，加载JVM自身工作类；

Extension classLoader，加载目录下类；

Application classLoader，加载classpath指定类。

JVM工作流程：类加载器—>字节码校验器—>翻译字节码—>编译执行
